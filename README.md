# Agency.io_project# Agency AI

A modern responsive digital agency website built with React, Vite, and Tailwind CSS.

## Features

- Responsive design
- Dark / Light mode
- Animated UI with Motion
- Services section
- Team section
- Portfolio / Our Work section
- Contact form using Web3Forms
- Toast notifications with React Hot Toast

## Tech Stack

- React
- Vite
- Tailwind CSS
- Motion
- React Hot Toast
- Web3Forms

## Installation

```bash
npm install
npm run dev
